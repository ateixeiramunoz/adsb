Cómo funciona

Servicios en este compose:

  • postgres: almacena posiciones (tabla positions) y metadatos (aircraft).
  • adsb_sender: colector remoto (lee dump1090 y envía a Postgres).
  • adsb_view_db: API + mapa (consume Postgres).
